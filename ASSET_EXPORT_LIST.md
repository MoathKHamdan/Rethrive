## üì¶ **Download Script**

Use this bash script to download all images (run after exporting from Figma Make):

```bash
#!/bin/bash

# Create directories
mkdir -p wordpress-assets/images/{experts,ecosystem,testimonials,case-studies,blog,clinicians,model,philosophy}

# Download logo
curl -o wordpress-assets/images/logo.png "https://iili.io/qHfdUsj.png"

# Download expert photos
curl -o wordpress-assets/images/experts/sarah-chen.png "https://iili.io/qHfxdP9.png"
curl -o wordpress-assets/images/experts/michael-rodriguez.png "https://iili.io/qHfz41t.png"
curl -o wordpress-assets/images/experts/emily-watson.jpg "https://iili.io/qHfOJpa.jpg"

curl -o wordpress-assets/images/experts/lisa-anderson.jpg "https://iili.io/qHfk3fp.jpg"

# Download ecosystem images
```
